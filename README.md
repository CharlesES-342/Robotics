# Robotics
## University - Year 3:
Controlling a robot to follow a path
Given a pre-defined motors.py file (consisting of all moving operations). This project aims to teach us a way of using a visual RGBD camera to track and monitor a physical environment, identify and follow a pre-defined path that will be made of a yellow peice of string. This track will change.
